# Download and format ICES area polygons

Download and format ICES area polygons

## Usage

``` r
prepareICESareas()
```

## Details

This function is an Ibrahim hack which does not work optimally because
the area polygons supplied on the ICES site are very complex and partly
corrupted. Use the premade polygons (`data(icesAreas)`) instead. These
polygons have been obtained from the ICES GIS team upon request and much
simpler.
