# R Package for Downloading Server-Side Data for BioticExplorer

This package can be used to download IMR Biotic database and to place it
into a [duckdb](https://r.duckdb.org/reference/duckdb.html) database,
providing access to all of the institute’s Biotic data from R.

## Details

The database can be downloaded and compiled using the
[`compileDatabase`](https://deepwaterimr.github.io/BioticExplorerServer/reference/compileDatabase.md)
function.
