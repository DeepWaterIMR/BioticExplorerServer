# Cruise series list

Use the
[`prepareCruiseSeriesList`](https://deepwaterimr.github.io/BioticExplorerServer/reference/prepareCruiseSeriesList.md)
function to update the list. Need to be updated every time data from the
database are downloaded.

## Usage

``` r
cruiseSeries
```

## Format

A data table

## Source

Institute of Marine Research (<https://imr.no>)
